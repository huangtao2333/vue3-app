📱 项目介绍

这是一个基于 Vue3 + TypeScript + Vite 构建的高性能外卖应用，采用现代化的前端技术栈，实现了完整的外卖业务流程。

Vue 3 TypeScript Vite Vant UI Pinia PWA

  • 购物车管理
  • 订单管理
  • 地址管理
  • 个人中心
